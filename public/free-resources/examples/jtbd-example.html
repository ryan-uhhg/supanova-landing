핵심 Job 3가지 — 고객이 진짜 원하는 것

"빠르게 분석 결과를 만들고 싶다"

고객 Pain Points — 기능·감정·사회적

기능적 Pain
  • 프레임워크 선택에 시간 소모
  • 분석 후 슬라이드 작성 이중 작업
  • 템플릿이 너무 딱딱하고 수정 불편
  • AI 결과물 품질 편차 예측 불가
감정적 Pain
  • 컨설팅 퀄리티 못 낸다는 열등감
  • 마감 전 분석이 미완성인 불안감
  • "이게 맞나?" 방법론 확신 부재
  • 남들은 빠른데 나만 느리다는 느낌
사회적 Pain
  • 팀원이 프레임워크를 이해 못 함
  • 클라이언트에게 설명 반복 피로
  • 회의에서 같은 언어 공유 안 됨
  • 경쟁자가 더 세련된 자료 사용

기회 발굴 — 과소 충족 vs 과잉 충족

과소 충족 (황금 기회)
  • 프레임워크 선택 자동 추천 — 없음
  • 산업별 벤치마크 데이터 내장 — 없음
  • 팀 공동 편집·댓글 — 없음
  • PDF 인쇄 최적화 슬라이드 — 희소
  • 한국어 맞춤 용어·사례 — 매우 희소
과잉 충족 (차별화 불필요)
  • 영문 템플릿 — 이미 넘침
  • 기본 SWOT 표 — 공짜 도구 다수
  • PPT 다운로드 형식 — 포화
  • 일반적 AI 요약 — ChatGPT 대체
Createnova 기회 포지셔닝

한국어 특화 + 고품질 디자인 + 프레임워크 자동 추천 = 국내 시장 독보적 포지션. 과소 충족 영역 집중 공략.

Job별 성공 지표 설계

기능적 Job 지표
  • 분석 완료 소요시간 (목표: ≤10분)
  • 첫 슬라이드 생성까지 시간
  • 재사용률 — 2회 이상 사용
  • 오류 없이 완성된 분석 비율
감정적 Job 지표
  • NPS 점수 (목표: ≥60)
  • "전문가 같다" 만족도 설문
  • SNS 공유·스크린샷 건수
  • 이탈률 — 결과 페이지 ≤20%
사회적 Job 지표
  • 팀 공유 건수 / 월
  • 초대를 통한 신규 가입률
  • 커뮤니티 언급·추천 수
  • 리퍼럴 전환율
우선 집중 지표 (North Star)

주간 활성 사용자(WAU) × 평균 분석 횟수 — "고객이 실제로 Job을 달성하는 횟수"
