#### S4. 요금제 FAQ (아코디언)

**Q1. 구독이 아닌 건 정말입니까?**
> 네. Supanova는 SaaS 구독 모델이 아닙니다. 한 번 결제하시면 영구 라이선스가 부여되며, 추가 비용은 일절 발생하지 않습니다. 모든 미래 업데이트도 무료로 제공됩니다.

**Q2. 얼리버드 특가는 언제 종료됩니까?**
> 선착순 50명 결제 완료 시점에 자동 종료됩니다. 종료 후 가격은 79,000원으로 복원됩니다. 현재 잔여 수량은 구매 페이지에서 실시간으로 확인하실 수 있습니다.

**Q3. Basic에서 Premium으로 업그레이드할 수 있습니까?**
> 네. Basic 구매 후 30일 이내에 Premium으로 업그레이드하시면 Basic 결제 금액을 전액 차감하여 차액만 결제하실 수 있습니다. 30일 초과 시에는 Premium 정가로 별도 구매하셔야 합니다.

**Q4. 여러 기기에서 사용할 수 있습니까?**
> Premium 라이선스는 1인 사용자 기준으로 최대 2대의 기기(예: 업무용 데스크톱 + 개인 노트북)에 설치하실 수 있습니다. 동시 사용이 아닌 1인 사용 조건입니다.

**Q5. 법인 대량 구매 할인이 있습니까?**
> 5라이선스 이상 대량 구매 시 별도 법인 할인을 제공합니다. support@supanova.co로 문의하여 주십시오.

**Q6. 환불 조건이 무엇입니까?**
> Premium 플랜에 한해, 귀하의 환경에서 앱이 정상 작동하지 않을 경우 구매일로부터 14일 이내에 100% 환불해 드립니다. 상세 조건은 [환불 정책](/refund)을 확인하여 주십시오.

**Q7. 결제 수단은 무엇을 지원합니까?**
> 신용카드(국내/해외), 카카오페이, 토스페이, 네이버페이를 지원합니다.

**Q8. 세금계산서 발급이 가능합니까?**
> 네. 법인 또는 사업자 구매 시 세금계산서를 발급해 드립니다. 결제 완료 후 support@supanova.co로 사업자등록증 사본과 함께 요청하여 주십시오.

#### S6. FAQ (보안 특화)

**Q1. 분석 시 인터넷 연결이 필요합니까?**
> 네. Supanova는 AI 분석에 Claude API(Anthropic)를 사용하므로 분석 실행 시 인터넷 연결이 필요합니다. 단, 전송되는 것은 파이썬이 로컬에서 추출한 텍스트만이며, 원본 파일(PDF, PPTX 등) 자체는 업로드되지 않습니다.

**Q2. 전송되는 텍스트에 민감 정보가 포함될 수 있지 않습니까?**
> 문서 내용에 따라 가능합니다. 이를 위해 Supanova의 전처리 모듈은 헤더, 푸터, 워터마크 등 불필요한 데이터를 사전 제거합니다. 또한 Anthropic의 API 이용약관에 따라 API를 통해 전송된 데이터는 모델 훈련에 사용되지 않습니다. 극도로 민감한 데이터(예: 특정 개인 정보)가 포함된 경우, 해당 문서의 투입 여부를 사전에 판단하시기를 권장합니다.

**Q3. Supanova 자체 서버에 데이터가 저장됩니까?**
> 아닙니다. Supanova는 자체 서버를 운영하지 않습니다. 귀하의 데이터는 오직 귀하의 기기 → Claude API 경로로만 이동하며, Supanova의 인프라를 경유하지 않습니다. 분석 결과는 귀하의 로컬 디스크에만 저장됩니다.

**Q4. 사내에서 운영하는 AI 서버(Azure OpenAI 등)에 연결할 수 있습니까?**
> 네. Supanova는 API 엔드포인트 설정을 변경하여 귀사의 사내 AI 인프라(Azure OpenAI, AWS Bedrock, 자체 호스팅 모델 등)에 직접 연결할 수 있습니다. 이 경우 추출된 텍스트도 사내망을 벗어나지 않으므로, 외부 API 호출 자체가 발생하지 않습니다. 가장 엄격한 보안 요건을 갖춘 금융·공공·군 환경에서도 도입이 가능합니다.

**Q5. IT 보안팀에 제출할 수 있는 기술 문서가 있습니까?**

### 5가지 시나리오

#### 시나리오 1: 신사업 기획 보고서

- **상황**: 대기업 전략기획팀 소속. 신사업 아이템 3건에 대한 타당성 분석 보고서를 3일 안에 경영진에게 보고해야 함.
- **기존 방식**: 각 아이템별 시장 보고서, 경쟁사 자료, 내부 재무 데이터를 수작업으로 읽고 요약. PowerPoint를 0에서 작성. 소요 시간: 각 아이템당 8시간 × 3건 = 24시간.
- **Supanova 적용**: 아이템당 관련 문서 5~7건을 투입 → 3C 분석 프레임워크 적용 → .pptx 출력. 소요 시간: 각 아이템당 10분 × 3건 = 30분.
- **핵심 산출물**: 3C 분석 기반 전략 보고서 .pptx (시장 기회, 자사 역량, 경쟁사 현황)

#### 시나리오 2: 경쟁사 심층 분석

- **상황**: 중견기업 마케팅팀. 주요 경쟁사 5개사의 최근 전략 동향을 분석하여 다음 분기 대응 전략을 수립해야 함.
- **기존 방식**: 뉴스 기사, 공시 자료, 업계 보고서를 개별적으로 스크랩하고 엑셀에 정리. 소요 시간: 2~3일.
- **Supanova 적용**: 경쟁사별 수집 문서를 투입 → Porter's Five Forces 적용 → 경쟁 환경 분석 보고서 출력. 소요 시간: 50분.
- **핵심 산출물**: Porter's Five Forces 기반 경쟁 분석 보고서

#### 시나리오 3: 투자 유치 IR 자료

- **상황**: 시리즈 A 투자 유치를 준비하는 스타트업 대표. 시장 규모, 경쟁 현황, 사업 모델, 재무 전망을 담은 IR 덱이 필요.
- **기존 방식**: VC 펀드 파트너 미팅까지 2주. 외부 컨설턴트 비용 300만원+, 납기 불확실.
- **Supanova 적용**: 내부 사업계획서 + 시장 보고서 투입 → McKinsey 7S 프레임워크 적용 → 투자자 보고용 .pptx 출력. 소요 시간: 1시간.
- **핵심 산출물**: 투자 유치용 IR 덱 .pptx

- **기존 방식**: 국가별 시장 보고서, 관세 자료, 현지 경쟁사 정보를 수집·번역·정리. 소요 시간: 1주.
- **Supanova 적용**: 국가별 문서를 투입 → SWOT 분석 적용 → 국가별 진입 전략 .pptx 출력. 소요 시간: 국가당 15분.

## Page 11: `/docs/faq` — 자주 묻는 질문

### FAQ 항목 (카테고리별)

#### 제품 일반

**Q. Supanova는 정확히 무엇입니까?**
> Supanova는 귀하의 랩탑에서 문서를 전처리한 뒤 Claude AI로 분석하여, 즉시 보고 가능한 .pptx 파일을 생성하는 프라이빗 인텔리전스 시스템입니다. 원본 파일은 업로드되지 않으며, 파이썬이 추출한 텍스트만 분석에 사용됩니다.

**Q. ChatGPT나 Claude 웹과 무엇이 다릅니까?**
> 핵심 차이는 3가지입니다: ① 원본 파일 비전송(파이썬이 텍스트만 추출하여 API로 전송), ② 문서 누적 기억(대화가 초기화되지 않음), ③ .pptx 다이렉트 출력(별도 정리 작업 불필요). 웹에서 직접 파일을 업로드하는 것과 달리, 원본 파일 자체가 외부로 나가지 않습니다.

**Q. 어떤 파일 포맷을 지원합니까?**
> PDF, DOCX, PPTX, XLSX, TXT, CSV를 지원합니다. 한 번에 최대 20개 파일을 투입할 수 있습니다.

**Q. 한국어 문서만 분석할 수 있습니까?**
> 아닙니다. 한국어, 영어, 일본어, 중국어 문서를 분석할 수 있습니다. 출력 언어는 프롬프트 설정으로 지정하실 수 있습니다.

**Q. GPU가 필요합니까?**
> 필요하지 않습니다. CPU만으로 작동하도록 최적화되어 있으며, 일반 업무용 노트북에서 원활하게 구동됩니다.

#### 구매 및 라이선스

**Q. 구독 모델입니까?**
> 아닙니다. 1회 결제 영구 라이선스입니다. 추가 비용이 발생하지 않습니다.

**Q. 여러 기기에서 사용할 수 있습니까?**
> 1인 사용자 기준 최대 2대의 기기에 설치하실 수 있습니다.

**Q. 팀/법인 라이선스가 있습니까?**
> 5라이선스 이상 구매 시 법인 할인을 제공합니다. support@supanova.co로 문의하여 주십시오.

**Q. 결제 후 어떻게 받습니까?**
> 결제 완료 즉시 등록하신 이메일로 다운로드 링크가 발송됩니다.

#### 보안

**Q. 원본 파일이 외부로 전송됩니까?**
> 아닙니다. 원본 파일(PDF, PPTX 등)은 귀하의 기기를 벗어나지 않습니다. 파이썬 전처리 모듈이 로컬에서 텍스트만 추출하고, 추출된 텍스트만 Claude API(Anthropic)로 전송됩니다. Supanova 자체 서버에는 어떤 데이터도 전송되지 않습니다. [보안 아키텍처 상세](/security)를 확인하여 주십시오.

**Q. 사내 보안 감사를 통과할 수 있습니까?**
> 원본 파일이 외부 전송되지 않으므로 DLP 정책 관점에서 유리합니다. 다만 텍스트 기반 API 호출이 발생하므로, 극도로 민감한 데이터(개인정보 등)가 포함된 경우 사내 보안팀과 사전 협의하시기를 권장합니다. 보안 검토용 기술 문서는 support@supanova.co로 요청하실 수 있습니다.

**Q. 사내 AI 서버(Azure OpenAI, AWS Bedrock 등)에 연결할 수 있습니까?**
> 네. Supanova의 설정에서 API 엔드포인트만 귀사의 사내 AI 서버 주소로 변경하면 즉시 연동됩니다. 이 경우 추출된 텍스트조차 사내망 밖으로 나가지 않아, 가장 엄격한 보안 환경에서도 사용할 수 있습니다.

#### 기술 지원

**Q. 기술 지원은 어떻게 받습니까?**
> Premium 구매자는 전용 지원 채널(이메일 우선 응답)을 이용하실 수 있습니다. Basic 구매자는 일반 이메일 지원을 제공합니다.

**Q. 앱이 작동하지 않으면 어떻게 합니까?**
> support@supanova.co로 문의하시면 24시간 이내에 기술 지원을 제공합니다. 해결 불가 시 [100% 환불 보증](/refund)이 적용됩니다.

#### S2. 프레임워크 아코디언 리스트

각 프레임워크: 접힌 상태에서 이름 + 한 줄 설명. 펼치면 상세 내용.

**1. McKinsey 7S**
- **한 줄**: 조직의 7가지 내부 요소를 분석하여 전략 정합성을 평가합니다.
- **상세**: Strategy(전략), Structure(조직 구조), Systems(시스템), Shared Values(공유 가치), Skills(역량), Style(리더십 스타일), Staff(인력) — 7개 요소의 상호 정합성을 분석하여 조직 변화 관리, 합병 후 통합, 전략 실행력 평가에 활용합니다.
- **적합한 상황**: 조직 진단, M&A 통합 분석, 전략 실행력 평가

**2. Porter's Five Forces**
- **한 줄**: 산업 내 5가지 경쟁 세력을 분석하여 수익성 구조를 파악합니다.
- **상세**: 기존 경쟁자, 잠재 진입자, 대체재, 공급자 교섭력, 구매자 교섭력 — 5개 세력의 강도를 분석하여 산업 매력도와 진입 전략을 도출합니다.
- **적합한 상황**: 시장 진입 전략, 산업 분석, 경쟁 환경 평가

**3. SWOT 분석**
- **한 줄**: 내부 강점/약점과 외부 기회/위협을 교차 분석합니다.
- **적합한 상황**: 전략 수립 초기, 사업 계획서, 연간 전략 리뷰

**4. 3C 분석**
- **한 줄**: 고객(Customer), 경쟁사(Competitor), 자사(Company)를 분석합니다.
- **적합한 상황**: 마케팅 전략, 신사업 기획, 시장 포지셔닝

**5. BCG Matrix**
- **한 줄**: 사업 포트폴리오를 시장성장률 × 시장점유율로 분류합니다.
- **적합한 상황**: 사업 포트폴리오 관리, 투자 우선순위 결정

**6. PESTEL 분석**
- **한 줄**: 거시 환경의 6가지 요인(정치, 경제, 사회, 기술, 환경, 법률)을 분석합니다.
- **적합한 상황**: 해외 시장 진출, 규제 환경 분석, 장기 전략

**7. Value Chain 분석**
- **한 줄**: 기업의 가치 창출 활동을 주활동과 지원활동으로 분해하여 경쟁 우위를 파악합니다.
- **적합한 상황**: 원가 구조 분석, 운영 효율화, 아웃소싱 의사결정

**8. Ansoff Matrix**
- **한 줄**: 제품 × 시장의 4가지 조합으로 성장 전략을 도출합니다.
- **적합한 상황**: 신사업 방향 설정, 성장 전략 수립

**9. Blue Ocean Strategy Canvas**
- **한 줄**: 경쟁 요소별 가치 곡선을 비교하여 차별화 전략을 시각화합니다.
- **적합한 상황**: 신제품 포지셔닝, 차별화 전략

**10. Kano Model**
- **한 줄**: 고객 기대 수준에 따라 기능을 필수/성능/매력 속성으로 분류합니다.
- **적합한 상황**: 제품 기획, 기능 우선순위 결정, UX 전략

**11. Business Model Canvas**
- **한 줄**: 사업 모델의 9개 구성 요소를 한 장으로 시각화합니다.
- **적합한 상황**: 사업 모델 설계, 스타트업 피칭, 사업 타당성 검토

**12. OKR Framework**
- **한 줄**: 목표(Objectives)와 핵심 결과(Key Results)를 구조화합니다.
- **적합한 상황**: 분기별 목표 설정, 팀 성과 관리, 전략 실행 추적
